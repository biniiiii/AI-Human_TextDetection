About Detect AI-Generated Text

A research project focused on developing fair and reliable AI-generated text detection systems using state-of-the-art transformer models.

The Problem We're Solving

Large Language Models (LLMs) like ChatGPT, Gemini, and DeepSeek have become widely used across various fields. However, their growing popularity raises concerns about potential misuse, particularly in academic settings where they can discourage genuine effort and undermine creativity.

Existing AI detection tools frequently exhibit bias against non-native English writers, resulting in false positive detections that unfairly penalize legitimate writers. Additionally, these tools are often vulnerable to simple paraphrasing techniques, compromising fair assessment.

Bridging the Research Gap

Despite the growing number of AI text detection systems, a clear research gap remains:

🌍

Non-Native Bias

Existing detectors are largely trained on native English datasets, reducing their reliability when evaluating texts written by non-native English speakers.

🔄

Paraphrase Evasion

Many current systems struggle to generalize across different AI models and are vulnerable to simple paraphrasing techniques.

⚖️

Fair Assessment

There is a need for detection systems that perform consistently across various AI-generated text sources while maintaining fairness.

Our Approach

1

Comprehensive Dataset Collection

We collected and integrated diverse datasets from platforms like HuggingFace, Kaggle, and GitHub, including AI-generated texts from multiple models (ChatGPT-4, Grok-3, Gemini) and human-written texts from various sources including non-native English essays.

2

Advanced Preprocessing Pipeline

Developed a refined preprocessing pipeline optimized for transformer models, including text cleaning, normalization, and filtering to ensure high-quality input while preserving semantic content.

3

BERT Fine-tuning

Fine-tuned BERT-base-uncased model using carefully selected hyperparameters (learning rate: 2e-5, epochs: 3, weight decay: 0.01) to achieve optimal performance on binary classification.

4

Rigorous Evaluation

Evaluated the model using multiple metrics including accuracy, precision, recall, F1-score, and confusion matrix to ensure reliable performance across different text types and sources.

Project Objectives

Main Objective

To develop an AI-generated text detection system using the BERT Transformer Model to accurately classify text as AI-generated or human-written.

Specific Objectives
  • Collect and integrate diverse datasets from multiple sources
  • Develop optimized preprocessing pipeline for transformer models
  • Train and fine-tune BERT for optimal binary classification
  • Minimize linguistic bias against non-native English speakers
  • Evaluate robustness against paraphrasing attacks
  • Deploy user-friendly web interface for real-time detection
